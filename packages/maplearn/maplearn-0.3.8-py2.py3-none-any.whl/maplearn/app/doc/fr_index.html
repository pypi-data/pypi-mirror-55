Bienvenu(e)

Mapping Learning (maplearn) vous aide à appliquer le Machine Learning sur vos jeux de données.

Que voulez-vous faire ?

  • Classification : prédire des valeurs discrètes (ou catégorielles), en utilisant des échantillons
  • Clustering : comme la classification, mais sans connaissance a priori (sans échantillons)
  • Régression : prédire des valeurs continues

Que voulez-vous faire précisément ?

maplearn vous guidera dans votre réflexion, en suivant 3 étapes :

Entrées/Sorties : pour choisir le/les fichiers sur lequel travailler.

Prétraitements : modifications à appliquer au jeu de données avant d'appliquer le machine learning. Exemple : réduire le nombre de dimensions.

Traitements : choisissez les algorithmes et comment vous allez les appliquer.
