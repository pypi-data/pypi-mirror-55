Install
-------

**via pip**

.. code-block:: bash

    pip install toad


**via source code**

.. code-block:: bash

    python setup.py install


Usage
-----

.. code-block:: python

    import toad


    data = pd.read_csv('test.csv')

    toad.detect(data)

    toad.quality(data, target = 'TARGET', iv_only = True)

    toad.IV(feature, target, method = 'dt', min_samples = 0.1)
